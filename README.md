# Railway-Reservation-System

A console-based railway reservation system developed using Core Java and Object-Oriented Programming concepts. This project simulates basic IRCTC functionalities such as train search, ticket booking, cancellation, and passenger management through a command-line interface.

---

## Features

- User Registration & Login
- Train Search
- Ticket Booking
- Ticket Cancellation
- Passenger Details Management
- Seat Availability Check
- Booking History
- Console-Based Interactive Menu

---

## Tech Stack

- Core Java
- OOP Concepts
- Collections Framework
- Exception Handling
